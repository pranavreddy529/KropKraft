# KropKraft Innovations - Agricultural Drone Technology Website

A modern, responsive marketing website for KropKraft Innovations, an Indian agricultural drone technology company. Features a premium, enterprise SaaS design with both light and dark mode variants.

## Features

### Design
- **Light & Dark Mode**: Toggle between light and dark themes with smooth transitions
- **Responsive Design**: Optimized for desktop (1440px), tablet (768px), and mobile (390px)
- **Modern UI**: Clean, premium enterprise SaaS style with card-based layouts
- **Consistent Spacing**: Built on an 8px grid system for visual harmony
- **Smooth Animations**: Hover effects, transitions, and scroll behavior
